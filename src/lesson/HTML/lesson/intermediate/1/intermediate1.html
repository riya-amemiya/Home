CSSはどこの、どれを、どう変えるか、を指定します

まずタグとクラスの指定の仕方の違いをみてみましょう

クラスの場合

クラスはHTMLで開始タグの中でclass=""で指定します

<p class=""></p>

このように書きます

CSSで指定するにはclass=""でつけた名前の先頭に.をつけます

例えばmainクラスを宣言した場合

HTML

<p class="main"></p>

CSS

.main{}

{}の中に書いていきます

タグの場合

タグの場合は.をつけずに指定します

ですが全てのタグに適応されるので共通のレイアウトに使います

CSSに触れてみる

CSSを書いてみましょう

色を変えてみましょう

まず適応する場所を指定します

(クラスとタグの指定の仕方に注意してください)

{}の中にcolor: red;と書いてください

色を変える場合はcolorを指定します
